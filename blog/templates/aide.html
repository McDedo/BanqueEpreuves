Centre d'Aide

Bienvenue sur Banque d'Épreuves, votre plateforme de vente de documents académiques téléchargeables. Cette page vous guide pour utiliser efficacement notre site.

1. Création de votre compte

Pour télécharger des documents, vous devez d'abord créer un compte :

  • Cliquez sur le lien « Connexion » en haut du site.
  • Puis cliquez sur « Créer un compte ».
  • Remplissez le formulaire d’inscription avec vos informations.
  • Cliquez sur « Connexion » une fois l’inscription terminée.

2. Télécharger un document

Après vous être connecté :

  • Parcourez les documents disponibles selon vos besoins.
  • Cliquez sur le bouton « Télécharger » situé sous le document de votre choix.
  • Si le document est payant, vous serez redirigé vers la page de paiement.

3. Moyens de paiement disponibles

Nous proposons deux options sécurisées pour régler vos achats :

  1. Mobile Money (Orange Money, Moov, MTN... selon votre pays).
  2. Carte bancaire (Visa, Mastercard, etc.).

4. Rechercher un document

Pour trouver un document spécifique :

  • Utilisez le module de recherche sur la page d’accueil.
  • Sélectionnez :
    • La catégorie du document,
    • Le niveau d’étude,
    • L’année,
    • La matière.
  • Puis cliquez sur « Rechercher ».

5. Besoin d’aide supplémentaire ?

Vous pouvez nous contacter à tout moment via la page Contact ou en envoyant un mail à : actucoursepreuves@gmail.com.
